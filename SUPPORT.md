# Support

Please include the following information when reporting an issue:

- Minecraft version
- Fabric Loader version
- Fabric API version
- Cobblemon version
- Cobblemon Chat Alerts version
- Whether the issue happens in single-player, LAN, or a dedicated server
- The relevant part of `logs/latest.log`
- Your `config/cobblemon-chat-alerts.json` file, if the issue is configuration-related

For gameplay issues, please describe the event that should have triggered an alert and what actually happened.

If the project is published on GitHub, please use the bug report template so the issue has enough context to reproduce.
